Named, Function Expression, and Arrow Function

I got this function from my Studio 2 project. The function change the time display's value and adjusted its visibility.

Named Function


            function showTime(point, data) {
                // Change the value
                timeDisp.innerHTML = `<p id="time">${data[point].time}</p>`;
        
                // Adjust the visibility
                timeDisp.classList = "vis";
                numDisp.classList = "invis";
                moodDisp.classList = "invis";
            }

Function Expression


            const showTime = function(point, data) {
                // Change the value
                timeDisp.innerHTML = `<p id="time">${data[point].time}</p>`;
        
                // Adjust the visibility
                timeDisp.classList = "vis";
                numDisp.classList = "invis";
                moodDisp.classList = "invis";
            }

Arrow Function

            
            const showTime = (point, data) => {
                // Change the value
                timeDisp.innerHTML = `<p id="time">${data[point].time}</p>`;
        
                // Adjust the visibility
                timeDisp.classList = "vis";
                numDisp.classList = "invis";
                moodDisp.classList = "invis";
            }
